# Результаты моделирования электрического поля точечных зарядов

## Теоретическая основа задачи

Электрическое поле системы точечных зарядов описывается принципом суперпозиции. Напряжённость электрического поля (\vec{E}) в произвольной точке пространства равна векторной сумме напряжённостей, создаваемых каждым зарядом:

[
\vec{E}(\vec{r}) = \sum_i k \frac{q_i}{r_i^3} \vec{r}_i,
]

где:

* (q_i) — величина i-го точечного заряда,
* (\vec{r}_i) — радиус-вектор от заряда до рассматриваемой точки,
* (r_i = |\vec{r}_i|),
* (k) — коэффициент пропорциональности (кулоновская константа).

Силовые линии электрического поля определяются как интегральные кривые векторного поля (\vec{E}). По определению, касательная к силовой линии в каждой её точке совпадает с направлением вектора напряжённости поля:

[
\frac{d\vec{r}}{ds} = \frac{\vec{E}}{|\vec{E}|},
]

где (s) — параметр вдоль линии.

---

## Численное моделирование

В ходе моделирования была реализована явная численная схема (метод Эйлера первого порядка) для построения силовых линий электрического поля. Интегрирование выполнялось в параметрической форме:

[
\begin{cases}
\dfrac{dx}{ds} = \dfrac{E_x}{|E|},\
\dfrac{dy}{ds} = \dfrac{E_y}{|E|}.
\end{cases}
]

Начальные точки силовых линий выбирались на малом расстоянии от зарядов и равномерно распределялись по углу. Направление интегрирования определялось знаком заряда:

* для положительного заряда линии выходят из него;
* для отрицательного — входят в него.

Интегрирование прекращалось при:

* выходе за границы расчётной области;
* приближении к другому заряду;
* уменьшении модуля напряжённости до пренебрежимо малого значения.

---

## Анализ полученных результатов

### Система разноимённых зарядов

Для системы положительного и отрицательного зарядов получена характерная картина электрического поля:

* силовые линии начинаются на положительном заряде и заканчиваются на отрицательном;
* линии распределены симметрично относительно оси, соединяющей заряды;
* плотность силовых линий возрастает в области между зарядами, что соответствует увеличению напряжённости поля.

Полученный результат полностью согласуется с теоретическими представлениями об электрическом поле диполя.

### Векторное поле и пробный заряд

Дополнительно была визуализирована векторная картина поля в виде стрелок, направление которых совпадает с направлением вектора напряжённости. Движение пробного заряда вдоль силовой линии подтверждает корректность построения интегральных кривых и соответствие численной модели физическому смыслу задачи.

## Выводы

В результате выполнения работы было проведено численное моделирование электрического поля системы точечных зарядов. Реализованная математическая модель на основе закона Кулона и принципа суперпозиции корректно описывает структуру электрического поля.

Построенные силовые линии соответствуют теоретическим определениям и физическим ожиданиям для систем одноимённых и разноимённых зарядов. Используемый численный метод обеспечивает устойчивое и наглядное воспроизведение геометрии электрического поля, что подтверждает корректность выбранного подхода и реализации алгоритма.
